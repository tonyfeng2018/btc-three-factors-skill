# BTC三因子评分参考手册（v2.0）

## 评分结构总览

| 因子 | 权重 | 满分 |
|------|------|------|
| 一、全球流动性与风险偏好 | 45% | 45分 |
| 二、机构采用 & ETF资金流入 | 30% | 30分 |
| 三、监管环境 & 衍生品 | 25% | 25分 |
| **合计** | **100%** | **100分** |

## 信号阈值

| 总分区间 | 信号 | 操作建议 |
|---------|------|---------|
| **≥ 72分** | 🟢 买入/加仓 | BTC短期上涨条件已满足，流动性充裕+ETF大额流入+情绪高涨，短期胜率极高 |
| **56-71分** | ⚪ 观望（持仓不动） | 市场有一定支撑，但不具备立刻爆发动能 |
| **26-55分** | 🟡 风险提示 | 动能衰退，短期不宜追多，建议对冲或部分止盈 |
| **≤ 25分** + 核心扣分项 | 🔴 卖出/减仓 | BTC短期下跌风险极高，明确卖出/减仓信号 |

> **核心扣分项**：ETF日净流出（-10）、美元指数强势反弹（-10）、美联储鹰派（-8）

### ⚠️ 动量崩盘规则（抢跑卖出）
如果总分在 **24小时内骤降 ≥ 30分**，无论绝对总分是多少：
→ 立即触发：**🚨 核心逻辑证伪，动量崩盘，立即卖出/对冲**

## 一、全球流动性与风险偏好（满分45分）

### 1.1 美元指数 DXY（满分15分）

| 条件 | 得分 |
|------|------|
| 持续走弱：连续3日下跌 **或** 周跌幅 > 1% **或** DXY < 100.0 | **+15分** |
| 小幅回落 / 偏弱震荡（周跌幅 0-1%） | **+8分** |
| 横盘 / 偏强震荡（周变化 ±1% 内） | **0分** |
| 强势反弹：连续上涨 **或** 周涨幅 > 1% **或** DXY > 103.0 | **-10分** |

**数据源**：DXY现货指数（Yahoo Finance / TradingView）

**操作**：
- `dxy_now`：当前 DXY 数值（如 98.8）
- `dxy_7d`：7天前 DXY 数值（用于计算周变化率）

---

### 1.2 VIX恐慌指数 + 美股风险偏好（满分12分）

| 条件 | 得分 |
|------|------|
| VIX < 18 **且** 纳斯达克或SPX当日上涨 > 0.5% | **+12分** |
| VIX 18-22 **或** 美股小幅波动（无明显方向） | **+6分** |
| VIX > 22 **或** 美股明显下跌（纳指/SPX当日跌 > 0.5%） | **0分** |

**数据源**：
- VIX：TradingView `CBOE Volatility Index (VIX)`
- 纳斯达克/标普：Yahoo Finance `^IXIC` / `^GSPC`

---

### 1.3 美联储政策预期（满分10分）

| 条件 | 得分 |
|------|------|
| 市场定价年内降息预期升温 **或** 官员明确鸽派/中性偏鸽表态 | **+10分** |
| 中性无明显表态 | **0分** |
| 加息预期升温 **或** 明确紧缩/鹰派信号 | **-8分** |

**观察信号**：
- CME FedWatch Tool：年内降息概率变化
- 官员讲话关键词：鲍威尔/威廉姆斯/古尔斯比等最新表态
- 特朗普推文或财政信号

**参数**：`fed_signal` = `"dovish"` / `"neutral"` / `"hawkish"`

---

### 1.4 全球流动性指标（满分8分）

| 条件 | 得分 |
|------|------|
| 最近7天美联储资产负债表扩张 **或** M2货币供应增加 | **+8分** |
| 无明显扩张信号 | **0分** |

**数据源**：FRED `M2SL` / 美联储总资产 `WALCL`

**参数**：`liquidity_signal` = `"expanding"` / `"neutral"`

---

## 二、机构采用 & ETF资金流入（满分30分）

### 2.1 US Spot BTC ETF 日/周净流入（满分18分）

| 条件 | 得分 |
|------|------|
| 日净流入 > $300M **或** 周净流入 > $1B | **+18分** |
| 日净流入 $100M-$300M | **+10分** |
| 日净流入 $0-$100M（或无数据/平淡日） | **0分** |
| 日净流出（含ETF大量换手/赎回） | **-10分** |

> ⚠️ **重要**：ETF已是BTC的**边际定价者**，日流入 > $300M 是强烈买入信号。
> 平淡日无新闻时默认0分，属正常现象，不代表负面。

**数据源**：Farside / SoSoValue / Bitwise ETF Dashboard

**参数**：
- `etf_daily_net`：日净流入（美元）
- `etf_weekly_net`：周净流入（美元）

---

### 2.2 机构/企业动态（满分8分）

| 条件 | 得分 |
|------|------|
| 当日或48小时内有重大正面公告（如MicroStrategy大额购买、企业宣布持仓、ETF获批等） | **+8分** |
| 无重大机构动态 | **0分** |

**观察**：MicroStrategy持仓变化、贝莱德/富达ETF持仓变动、企业/国家宣布买入BTC

**数据源**：Arkham / Whale Alert / 公司公告 / Twitter

**参数**：`has_inst_news` = `True` / `False`

---

### 2.3 稳定币供应增长（满分4分）

| 条件 | 得分 |
|------|------|
| 7天内USDT+USDC总供应量增加 > $1B | **+4分** |
| 持平或萎缩 | **0分** |

**逻辑**：稳定币是币圈"流动性弹药"，供应扩张代表增量资金准备入场。

**数据源**：Dune Analytics `stablecoin supply` / CryptoQuant

**参数**：`stablecoin_change_7d`：7天供应变化（美元）

## 三、监管环境 & 衍生品（满分25分）

### 3.1 监管新闻（满分12分）

| 条件 | 得分 |
|------|------|
| 当日或48小时内有**明确利好**进展（SEC批准ETF、立法通过、监管清晰化） | **+12分** |
| 谈判进行中，无突破也无负面 | **+6分** |
| 负面进展（SEC起诉、禁令、监管打压） | **0分** |

**观察**：SEC/ CFTC / 立法机构最新动态，Trump加密政策，特朗普关于Crypto的推文

**参数**：`reg_status` = `"bullish"` / `"neutral"` / `"bearish"`

---

### 3.2 On-chain供给信号（满分8分）

| 条件 | 得分 |
|------|------|
| 交易所BTC储备**持续下降**（聪明钱锁仓）**或** 长期持有者（LTH）供给 > 14.5M BTC且稳定（持币不卖） | **+8分** |
| 持平或上升（持币者卖出迹象） | **0分** |

**逻辑**：BTC从交易所流出 = 投资者长期持有意愿强，抛压小

**数据源**：Glassnode / CryptoQuant `Exchange Reserve` / `LTH Supply`

**参数**：`onchain_signal` = `"bullish"` / `"neutral"`

---

### 3.3 衍生品微观结构（满分5分）

| 条件 | 得分 |
|------|------|
| 资金费率处于**健康区间**（年化 -1% 至 +1% 之间，无极端多头过热）**且** 现货CVD（累计成交量_delta）买盘偏强 | **+5分** |
| 资金费率健康但CVD中性 | **+3分** |
| 资金费率极度狂热（年化 > 3%，多头过热）或 CVD卖盘主导 | **0分** |

**逻辑**：资金费率过热往往是散户多头接盘信号，随后容易踩踏。

**数据源**：Coinglass / 交易所（币安/OKX/Bybit）资金费率

**参数**：
- `funding_rate`：永续合约资金费率（年化，如 0.001 = 0.1%）
- `cvd_signal` = `"buy"` / `"neutral"` / `"sell"`

## 数据获取优先级

当需要实时数据时，按以下优先级获取：

### 必须数据（DXY、VIX、ETF）
```
DXY：   curl "https://query1.finance.yahoo.com/v8/finance/chart/UBT?interval=1d&range=10d"
VIX：   curl "https://query1.finance.yahoo.com/v8/finance/chart/%5EVIX?interval=1d&range=5d"
纳指：  curl "https://query1.finance.yahoo.com/v8/finance/chart/%5EIXIC?interval=1d&range=5d"
SP500： curl "https://query1.finance.yahoo.com/v8/finance/chart/%5EGSPC?interval=1d&range=5d"
```

```
Farside:     https://farside.co.uk/bitcoin/
SoSoValue:   https://sosovalue.com/
Bitwise:     https://bitwise.co/
```

### On-chain数据
```
Glassnode:   https://glassnode.com
CryptoQuant:  https://cryptoquant.com
```

## 示例评分场景

### 场景A：正常偏多市场
```
输入：dxy=98.8, vix=16, etf_daily_net=$500M, fed_signal=dovish, reg_status=bullish
结果：
  因子1(DXY+VIX+Fed+Liq)：+15+6+10+0 = 31/45
  因子2(ETF+Inst+Stable)：+18+0+0 = 18/30
  因子3(Reg+Onchain+Deriv)：+12+0+0 = 12/25
  总分：61分 → ⚪ 观望
```

### 场景B：流动性危机（美股暴跌+ETF流出）
```
输入：dxy=104.5, vix=38, etf_daily_net=-$400M, fed_signal=hawkish, reg_status=bearish
结果：
  因子1：-10+0-8+0 = -18/45
  因子2：-10+0+0 = -10/30
  因子3：0+0+0 = 0/25
  总分：-28分 → 🔴 卖出（≤25分且含核心扣分项）
  动量崩盘：如24h前为50分，骤降78分 → 🚨 立即卖出
```

### 场景C：全面利好（降息预期+ETF大流入+监管利好）
```
输入：dxy=97.5, vix=14, etf_daily_net=$800M, fed_signal=dovish, reg_status=bullish, liquidity_signal=expanding
结果：
  因子1(DXY 15 + VIX 12 + Fed 10 + Liq 8)：+15+12+10+8 = 45/45
  因子2(ETF 18 + Inst 8 + Stable 4)：+18+8+4 = 30/30
  因子3(Reg 12 + Onchain 8 + Deriv 5)：+12+8+5 = 25/25
  总分：100分 → 🟢 极强买入
```
